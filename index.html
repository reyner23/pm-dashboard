Storage mode — currently using Local browser storage

By default PM Hub saves everything in your browser's local storage — fast, offline, no setup. To share real-time with your team, connect Firebase (free, 5 min setup).

Firebase setup (5 min)
1
Create a free Firebase project
Go to console.firebase.google.com → Create project → disable Google Analytics → Create
2
Enable Firestore
In your project → Build → Firestore Database → Create database → Start in test mode → choose your region → Enable
3
Register a web app
Project settings (gear icon) → Your apps → Web app icon (</>) → Register app → copy the firebaseConfig object values below
